# ADR-0004: Project Name

## Context

The goal was a name for the agentic OS that didn't already exist. Research showed that obvious names like "AgentOS" are already taken multiple times over (e.g. agentos.org, agentos.sh, several GitHub projects).

## Decision

Working name: **NyxelOS** — a combination of "Hearth" (your own home, standing for self-hosting) and "Loom" (weaving frame, standing for weaving together chats, agents, skills, MCP servers, and the knowledge base). Research found no existing projects under this name.

## Alternatives

"Wovenmind" and "Kestrion" were also checked and showed no collisions, but were judged less fitting for the self-hosting metaphor and kept as backup names.

## Open Item

Before a public launch, a proper domain, npm package, and trademark search still needs to happen (only a web search was performed, not a binding trademark check). Status is therefore "provisional" until that is confirmed.

# ADR-0004: Project Name

## Context

The goal was a name for the agentic OS that didn't already exist. Research showed that obvious names like "AgentOS" are already taken multiple times over (e.g. agentos.org, agentos.sh, several GitHub projects).

## Decision

Working name: **NyxelOS**. The first candidate, "Hearthloom" (a metaphor combining "Hearth" for self-hosting and "Loom" for weaving together chats, agents, skills, MCP servers, and the knowledge base), was rejected after review — it didn't land as a brand. A follow-up round asked for a short, invented, infrastructure-sounding word instead of a metaphor. "Nyxel", "Quorin", "Draveon", and "Ilvara" were shortlisted after a collision check found no existing open source projects, products, or notable accounts under any of them. "Nyxel" (used as "NyxelOS") was picked as the final name.

## Alternatives

"Quorin" and "Ilvara" were also checked and showed no collisions; they remain backup names if "NyxelOS" collides during a later domain/trademark check. "Hearthloom", "Vantrix", "Halvex", and "Draveon" were considered and dropped along the way (either for not matching the desired tech-brand sound, or for colliding with existing small unrelated projects).

## Open Item

Before a public launch, a proper domain, npm package, and trademark search still needs to happen (only a web search was performed, not a binding trademark check). Status is therefore "provisional" until that is confirmed.
